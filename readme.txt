
1) IMPORT PersonalHealthManagement.war as a war file in eclipse environment

2) Add a webserver(apache tomcat) to the runtime environment of eclipse

	 . Eclipse->Preferences->Server->Runtime Environments->add (choose the installation directory of the webserver)

3) Add the webserver to the Project 

	Right click on project properties -> project Facets -> runtimes-> add the server



Technical Stack:
UI:JSP
Backend:Java
IDE:Eclipse
